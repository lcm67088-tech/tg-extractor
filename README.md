# 텔레그램 링크 추출기 (Python GUI)

텔레그램 HTML 파일에서 채널/그룹 링크를 추출하고 정보를 조회하는 데스크탑 앱

## 실행 방법

```bash
python main.py
```

## 요구사항
- Python 3.8 이상
- 별도 패키지 설치 불필요 (표준 라이브러리만 사용)
- tkinter 포함 버전 필요 (보통 기본 포함)

> Windows에서 tkinter 없을 경우: Python 설치 시 "tcl/tk and IDLE" 체크 필요

## 기능

### 링크 추출
- HTML 파일 다중 업로드 (드래그 앤 드롭 or 파일 선택)
- 텍스트 직접 입력으로 링크 추가
- 초대링크(`t.me/+`) / 공개채널 자동 분류
- 중복 자동 제거

### 정보 조회
- 더블클릭 or 우클릭 → 단일 정보 조회
- 일괄 조회: 전체 / 초대링크만 / 공개채널만
- 3개 병렬 조회로 빠른 처리
- 제목, 멤버수, 타입, 설명 표시

### 내보내기
- TXT 다운로드 (URL만)
- CSV 다운로드 (전체 정보 포함, UTF-8 BOM)
- 전체 복사 (클립보드)

### 필터
- 검색 (URL/제목/설명/출처 통합 검색)
- 탭 필터 (전체/초대링크/공개채널)
- 중복 제거 토글
